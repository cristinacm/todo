Domain Module for Task-Oriented Dialogue management Ontology (TODO): Description back to ToC

The TODO Domain module deals with the concepts used for domain modelling, in terms of the actions that the target system can perform and the elements that are present in the scenario. Most importantly, this module is in charge of obtaining a target-system-readable command from a natural interaction.
