Dialogue Module for Task-Oriented Dialogue management Ontology (TODO): Description back to ToC

The TODO Dialogue module deals with the concepts used for dialogue modelling, both for dialogue management and dialogue tracing. For this module, the knowledge covering the former is intended to be implemented as static, and the latter as dynamic, as it will be instantiated during the dialogue process.
